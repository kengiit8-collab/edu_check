# ระบบเช็คชื่อนักเรียนออนไลน์ (edu_check)

โปรเจกต์นี้เป็นระบบเช็คชื่อนักเรียนออนไลน์ด้วย PHP และ MySQL เหมาะสำหรับติดตั้งบน MAMP/XAMPP หรือเครื่องเซิร์ฟเวอร์ที่รองรับ PHP + MySQL

## ไฟล์สำคัญ

- `index.php` หน้าสรุปภาพรวมการเช็คชื่อทุกห้อง
- `login.php` หน้าเข้าสู่ระบบของผู้ดูแลระบบ ครู และนักเรียน/ผู้ปกครอง
- `admin.php` หน้าจัดการข้อมูลและเช็คชื่อนักเรียน
- `check_summary.php` หน้าสรุปผลการเช็คชื่อ
- `homeroom_sheet.php` รายงานตามห้องเรียน
- `student_attendance.php` หน้าดูข้อมูลการมาเรียนของนักเรียน
- `config.php` ตั้งค่าฐานข้อมูล
- `sql/edu_check.sql` ไฟล์ฐานข้อมูลพร้อมข้อมูลเริ่มต้น
- `sql/seed_demo.sql` ไฟล์ข้อมูลตัวอย่างสำหรับรีเซ็ตข้อมูลทดลอง
- `assets/style.css` ไฟล์ตกแต่งหน้าจอ
- `img/` โฟลเดอร์รูปภาพ/โลโก้

## ความต้องการของระบบ

- PHP 7.4 ขึ้นไป แนะนำ PHP 8.x
- MySQL หรือ MariaDB
- เปิดใช้งานส่วนเสริม PHP: `mysqli`, `ZipArchive`, `SimpleXML`
- เว็บเซิร์ฟเวอร์ เช่น Apache บน MAMP หรือ XAMPP

## วิธีติดตั้งบน MAMP

1. คัดลอกโฟลเดอร์ `edu_check` ไปไว้ที่ `C:\MAMP\htdocs\edu_check`
2. เปิด MAMP แล้ว Start Servers
3. เปิด phpMyAdmin ที่ `http://localhost/phpMyAdmin`
4. สร้างฐานข้อมูลชื่อ `edu_check`
5. เลือกฐานข้อมูล `edu_check` แล้ว Import ไฟล์ `sql/edu_check.sql`
6. ตรวจสอบไฟล์ `config.php` ให้ตรงกับเครื่องที่ใช้

```php
define('DB_HOST', 'localhost');
define('DB_USER', 'root');
define('DB_PASS', 'root');
define('DB_NAME', 'edu_check');
```

7. เปิดระบบที่ `http://localhost/edu_check`

## การเข้าสู่ระบบ

ระบบมี 3 ประเภทผู้ใช้

- ผู้ดูแลระบบ: เข้า `login.php?type=admin`
- ครูประจำชั้น: เข้า `login.php?type=teacher`
- นักเรียน/ผู้ปกครอง: เข้า `login.php?type=student`

บัญชีเริ่มต้นที่มีในฐานข้อมูล:

| ประเภท | ชื่อผู้ใช้ | หมายเหตุ |
| --- | --- | --- |
| ผู้ดูแลระบบ | `admin` | ใช้จัดการข้อมูลทั้งหมด |
| ครู | `m101` | ครูประจำชั้น ม.1/1 |
| ครู | `m201` | ครูประจำชั้น ม.2/1 |
| ครู | `m301` | ครูประจำชั้น ม.3/1 |
| นักเรียน | ใช้รหัสนักเรียน เช่น `D101001` | รหัสผ่านใช้รหัสนักเรียนเดียวกัน |

หลังติดตั้งเสร็จ แนะนำให้ผู้ดูแลระบบเปลี่ยนรหัสผ่านทันทีจากเมนูเปลี่ยนรหัสผ่านหรือเมนูผู้ใช้

## วิธีใช้งานสำหรับผู้ดูแลระบบ

1. เข้าสู่ระบบด้วยบัญชีผู้ดูแลระบบ
2. ไปที่เมนูตั้งค่าเพื่อกำหนดชื่อโรงเรียน ปีการศึกษา ภาคเรียน และโลโก้
3. จัดการข้อมูลพื้นฐาน:
   - ครู
   - ห้องเรียน
   - นักเรียน
   - รายวิชา
   - ตารางสอน
   - ผู้ใช้ระบบ
4. สามารถนำเข้าข้อมูลครูและนักเรียนด้วยไฟล์ Excel `.xlsx`
5. เข้าเมนูเช็คชื่อ เลือกวันที่และห้องเรียน แล้วบันทึกสถานะนักเรียน
6. ดูรายงานจากเมนูรายงาน สรุปเช็คชื่อ และรายงานตามห้องเรียน

## วิธีใช้งานสำหรับครู

1. เข้าสู่ระบบที่ `login.php?type=teacher`
2. ระบบจะแสดงห้องเรียนที่ครูรับผิดชอบ
3. เลือกวันที่และโหลดรายชื่อนักเรียน
4. เลือกสถานะของนักเรียนแต่ละคน:
   - มา
   - ขาด
   - ลา
   - มาสาย
   - กิจกรรม
5. ใส่หมายเหตุรายคนหรือหมายเหตุทั้งห้องได้
6. กดบันทึกเช็คชื่อ
7. หากต้องแก้ไข ให้เปิดรายการเดิมแล้วกดแก้ไขการเช็คชื่อ

## วิธีใช้งานสำหรับนักเรียน/ผู้ปกครอง

1. เข้าหน้า `login.php?type=student`
2. กรอกรหัสนักเรียนในช่องชื่อผู้ใช้
3. กรอกรหัสผ่านเป็นรหัสนักเรียนเดียวกัน
4. ระบบจะแสดงประวัติการมาเรียนของนักเรียนคนนั้นตามปีการศึกษาและภาคเรียน

## การสำรองและย้ายระบบ

1. สำรองไฟล์ทั้งโฟลเดอร์ `edu_check`
2. Export ฐานข้อมูล `edu_check` จาก phpMyAdmin เป็นไฟล์ `.sql`
3. เมื่อติดตั้งบนเครื่องใหม่ ให้นำไฟล์โฟลเดอร์ไปวางใน `htdocs`
4. Import ฐานข้อมูลกลับเข้า MySQL
5. แก้ `config.php` ให้ตรงกับรหัสผ่านฐานข้อมูลของเครื่องใหม่

## การดาวน์โหลดจาก GitHub

เมื่ออัปโหลดขึ้น GitHub แล้ว ผู้ที่มีลิงก์สามารถดาวน์โหลดได้จากปุ่ม `Code` > `Download ZIP`

ถ้า repository เป็นแบบ Public ทุกคนที่มีลิงก์สามารถดูและดาวน์โหลดได้ แต่ไม่สามารถแก้ไขไฟล์ใน repository ได้ ยกเว้นเจ้าของ repo หรือคนที่ได้รับสิทธิ์ Collaborator

ถ้าต้องการให้เฉพาะคนที่ได้รับลิงก์ดาวน์โหลดได้จริง ๆ โดยไม่เปิดสาธารณะทั้งเว็บ ให้ใช้ repository แบบ Private แล้วเพิ่มสิทธิ์เฉพาะบัญชี GitHub ที่ต้องการ

## หมายเหตุด้านความปลอดภัย

- เปลี่ยนรหัสผ่านผู้ดูแลระบบและครูทันทีหลังติดตั้ง
- อย่าใช้รหัสผ่านฐานข้อมูลเริ่มต้นบนเซิร์ฟเวอร์จริง
- จำกัดสิทธิ์โฟลเดอร์อัปโหลดรูปภาพให้เหมาะสม
- สำรองฐานข้อมูลเป็นประจำ
- หากเปิดใช้งานบนอินเทอร์เน็ตจริง ควรใช้ HTTPS
